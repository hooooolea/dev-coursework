# 智能警务管理系统 UI 规范

## 允许 / 禁止对照表

| 项目 | 允许 | 禁止 |
|------|------|------|
| 图标 | Element Plus 线性图标（el-icon） | Emoji 表情符号 |
| 颜色 | 主色 #1a237e / #1565c0，灰色系 | 渐变背景、彩虹配色、多色卡片 |
| 状态标签 | 仅两色：primary（正常）/ danger（异常） | 多色 tag（warning/success/info 混用） |
| 文字 | 黑体、宋体、系统字体 | 彩色文字、描边文字 |
| 卡片 | 白底、细灰边框、无阴影或极轻阴影 | 彩色边框、渐变卡片、装饰性色条 |
| 按钮 | 主操作用 primary，次操作用默认灰 | 五颜六色按钮 |
| 图表 | 单色或双色（蓝+灰） | 多色饼图、霓虹配色 |
| 布局 | 留白充足、对齐严格 | 堆砌装饰元素 |
| 动画 | 无或极短（<150ms） | 复杂过渡、闪烁效果 |
| Logo区 | 文字 + 线性图标 | Emoji 作为 Logo |

## 色彩规范

```
主色:      #1a237e  （深蓝，导航/标题）
次主色:    #1565c0  （中蓝，按钮/链接）
背景:      #f5f6f8  （页面底色）
卡片背景:  #ffffff
边框:      #e4e7ed
文字主色:  #303133
文字次色:  #606266
文字辅色:  #909399
状态-危险: #f56c6c（在逃/特急/布控）
状态-正常: #1565c0（正常状态，复用主色）
```

## 状态标签规范

仅用两种视觉权重：
- **需注意**（红底白字）：在逃、特急、布控中、待处置
- **普通**（灰底深字）：其他所有状态

## 字体规范

```
字号：页面标题 16px / 表格内容 13px / 辅助说明 12px
字重：标题 600 / 正文 400
行高：1.5
```
