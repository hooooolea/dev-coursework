## 阶段 2：zhangming 接警处置 + 案件侦查

**账号**：`zhangming / 123456`（v6.5 新增演示账号，**注意 authorities=[]，能登录但权限全放行**）

### 步骤 4：接警处置
1. 退出 admin → 登录 zhangming
2. 警情驾驶舱首页 → 看 **"待我处置 1"** 红色高亮
3. 左侧菜单"**报警受理**"
4. 顶部切"**我的警情**" Tab → 看到 1-2 条分配给自己的
5. 行操作列点红色"**现场打卡**" 图标 → 弹"操作成功"（status 2→3）

### 步骤 6：案件侦查
1. 左侧菜单"**案件管理**"
2. 找到 admin 升级生成的 **AJ20260615001**（状态"侦查中"）
3. 点行操作列蓝色"**案件详情**" → 右侧弹出抽屉
4. 抽屉含 3 Tab：
   - **侦查进展** → 点底部"**新增进展**" → 填"内容 / 下一步" → 提交
   - **证据材料** → 点底部"**新增证据**" → 选类型 + 写名称 → 提交
   - **嫌疑人** → 点底部"**新增嫌疑人**" → 选角色 + 写姓名 → 提交
5. 抽屉底部固定 3 按钮：**新增嫌疑人**（蓝）+ **审批通过**（绿）+ **移送检察院**（橙）
6. **注意**："审批通过"按钮目前是纯 UI 弹提示（不动 status），**这是演示系统 BUG**
