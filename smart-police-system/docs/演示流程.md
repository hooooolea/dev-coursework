## 步骤 1：接警录入 + AI 装备推荐

**操作**：admin 登录 → 报警受理 → 点击 **演示流程**

**自动执行**：
1. 弹窗自动填入斗殴警情（海淀区中关村南大街5号）
2. 提交接警 → 生成警情编号
3. 1.5 秒后自动派发给 **张建国（P001）**
4. AI 装备推荐卡片出现（必带：执法记录仪/对讲机/手铐/警棍，建议：防刺背心/手电/无人机/急救包）

**验证点**：卡片显示分级装备清单，确认 AI 已配置

## 步骤 2：接警处置（张建国登录）

**操作**：退出 → 登录 `zhangsan / 123456`

**验证点**：
- 首页 **待我处置** 红色高亮，数字 ≥ 1
- 报警受理列表可看到派发给自己的警情
- 点击 **处置中** 按钮可更新状态
- 点击 **关闭** 填写处置说明

## 步骤 3：升级案件

**操作**：张建国在报警列表点击 **升级案件**

**验证点**：
- 自动生成案件编号（AJ + 日期 + 序号）
- 关联原警情信息
- 进入案件管理可看到新案件

---

## 步骤 4：案件侦查

**操作**：张建国进入案件管理 → 点击案件

**验证点**：
- **进展**：可添加侦查进展记录
- **证据**：可添加证据材料（上传文件/填写描述）
- **嫌疑人**：可关联人员档案中的人员，标注嫌疑程度
- **结案**：可变更案件状态

## 步骤 6：统计分析（陈刚登录）

**操作**：退出 → 登录 `chengang / 123456`

**验证点**：

## 步骤 7：AI 警情研判

**操作**：陈刚 → 侧边栏 **AI警情研判**

**验证点**：
- 顶部 4 个统计卡片（近30天接警/在办案件/破案率/在岗警员）
- 左侧 AI 对话：点击快捷问题或输入分析需求
- AI 流式输出分析报告（需配置 MiMo API Key）
- 右侧图表：警情类型分布 + 区域分布
- 语音录入可用
