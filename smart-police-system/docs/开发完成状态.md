## AI 模块

| 功能 | 端点 | 说明 |
|------|------|------|
| API Key 配置 | `POST /api/ai/config/key` | Redis 存储，前端配置页 |

| 图片描述 | `POST /api/ai/omni/describe-image` | 证据照片 AI 标注 |
| 视频分析 | `POST /api/ai/omni/analyze-video` | 巡逻视频 AI 分析 |
| 语音识别 | `POST /api/ai/asr/transcribe` | 音频→文字 |

| 语音合成 | `POST /api/ai/tts/speak` | 文字→语音播报 |
| 报警装备推荐 | `GET /api/alarm/{id}/equipment-recommend` | 接警后自动触发 |

## MiMo API 配置

- Base URL: `https://token-plan-cn.xiaomimimo.com/v1`
- Auth: `Authorization: Bearer <key>`（OpenAI 兼容格式）
- Key 类型: Token Plan（`tp-` 开头）
- Key 存储: Redis `ai:mimo:api-key`

- 后端: 121 个 Java 文件, 123 个 API
- 前端: 38 个 Vue/JS 文件, 15 个页面 + 3 个组件
- SQL: 4 个脚本 (建表/初始数据/AI表/演示数据)
- 文档: 6 份设计文档 + 大作业报告 + PPT 结构

## 已修复的问题

| 问题 | 解决 |
|------|------|
| Java 25 + Lombok 不兼容 | JDK 21 |
| MyBatis-Plus @TableLogic 缺 is_deleted 列 | ALTER TABLE 补充 7 张表 |
| 表格 fixed="right" 右侧空白 | 改为 min-width |
| 权限缺失 (sys:role:view 等) | 补充权限记录 + 5 角色分配 |
| LoginVO @Builder 编译失败 | @AllArgsConstructor |
| BCrypt 密码不匹配 | 重新生成 123456 哈希 |
| MiMo Token Plan URL 错误 | 改为 token-plan-cn.xiaomimimo.com |
| MiMo Auth header 错误 | api-key → Authorization: Bearer |
| SSE + @PreAuthorize 401 | 去掉 SSE 端点的 @PreAuthorize |
| 环比/同比不刷新 | calcComparison 空函数修复 |
| 页面切换数据丢失 | 去掉 router-view mode="out-in" |
| 字体过小 | 全局 +1px，ECharts fontSize 统一 |
| 英文类型码 | 后端字典翻译 + 前端 label 函数 |
